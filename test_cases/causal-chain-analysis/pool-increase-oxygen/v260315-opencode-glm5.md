# 因果链分析报告

## 初始缺点

气泡中的氧气在逸出水面时大部分仍未溶解到水中

## 因果链分析

- 气泡中的氧气在逸出水面时大部分仍未溶解到水中
  - 氧气在气泡停留期间传递到水中的总量占气泡含氧量的比例低
    - 氧气传递效率低（×）
      - 液膜传质系数 k_L 低
        - 气泡表面的液膜边界层厚度大
          - 气泡表面的水流速度低
            - 水体整体湍流强度低
              - 池塘/湖泊水体几乎静止
                - 池塘/湖泊所处地形平坦
                - 池塘/湖泊远离河流入水口
              - 搅拌设备未配置
            - 气泡尾流扰动效应弱
              - 气泡空间分布稀疏
                - 曝气孔数量少
                - 曝气孔间距大
      - 气泡比表面积 A/V 小
        - 气泡直径大
          - 气泡生成时的初始直径大
            - 曝气孔直径大
          - 气泡在上升过程中直径增大（or）
            - 气泡膨胀
              - 静水压力低于气泡生成位置的压力
                - 气泡上方水深持续减小
                  - 气泡持续向上运动
                    - 空气密度小于水密度
            - 多个气泡相互合并
              - 多个气泡在上升路径上相遇
                - 曝气孔的平面分布密度高
              - 相邻气泡间的水膜破裂
                - 气泡相互接触
                - 气泡间液膜厚度低于临界值
                  - 气泡表面张力驱动液膜排开
      - 传质驱动力 ΔC 小（or）
        - 饱和溶解氧浓度 C* 低
          - 水温高
            - 夏季气温高
            - 太阳辐射加热水体
        - 水中实际溶解氧浓度 C 高
          - 藻类光合作用产氧
          - 水中存在之前曝气积累的溶解氧
        - 气泡内氧气分压低
          - 曝气气体为空气
            - 空气中氧气摩尔分数约为21%
      - 气泡在水中的停留时间 t 短
        - 气泡上升速度快
          - 气泡直径大（上方已分析，大气泡终端上升速度更高）
          - 气泡密度小于水密度
            - 空气的密度低
        - 水体深度小
          - 池塘/湖泊底部到水面的垂直距离短
            - 池塘/湖泊属于浅水体
              - 自然水体形态特征为面积大而深度小
