# 因果链分析报告

## 初始缺点

工业化养鱼中空气增氧方法的运营成本高。

## 因果链分析

* 工业化养鱼中空气增氧方法的运营成本高
  * 电能消耗大
    * 空气泵功率高
      * 空气注入量大
        * 空气溶解效率低
          * 气泡在水中的停留时间短
            * 气泡上升速度快
              * 气泡直径大
                * 气泵产生的气泡尺寸大
                * 气泡尺寸控制机制缺失
              * 水体深度浅
                * 池塘/湖泊深度有限
              * 水体湍流弱
                * 有效混合装置不存在
            * 气泡与水体接触面积小
              * 气泡直径大
              * 气泡形状接近球形
          * 氧气传输速率低
            * 氧气在水中的溶解度低
              * 水温高
              * 水体盐度高
            * 气液界面更新慢
              * 气泡运动轨迹单一
              * 界面扰动不足
          * 空气利用率低
            * 气泡溶解不完全
            * 气泡逸出
            * 氮气等不参与溶解的气体比例高
        * 鱼类耗氧率高
        * 空气中氧气比例低
      * 气泵效率低
        * 气泵设计参数与水产养殖需求不匹配
        * 气泵磨损
          * 气泵运行时间长
    * 运行时间长
      * 供氧需求持续
        * 鱼类呼吸需求持续
        * 水体自然复氧能力不足
  * 设备维护成本高
    * 管道系统长度长
      * 管道覆盖面积大
    * 管道系统分支多
    * 管道堵塞频率高
      * 水体杂质含量高
      * 生物膜生长
    * 管道腐蚀速率高
      * 水体化学腐蚀性强
      * 材料抗腐蚀性差
    * 气泵故障率高
      * 环境湿度高
      * 水中颗粒物浓度高
      * 定期维护执行率低
  * 初始投资成本高
    * 大功率空气泵价格高
    * 管道系统材料成本高
    * 安装工程成本高
