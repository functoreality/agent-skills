# 因果链分析报告

## 初始缺点

纸张从白板上掉落。

## 因果链分析

* 纸张从白板上掉落（or）
  * 纸张与白板之间的摩擦力小于纸张重力
    * 纸张与白板之间的摩擦力小（or）
      * 纸张与白板之间的正压力小（or）
        * 磁铁与白板之间的磁力小（×）
          * 磁铁磁性强度弱（or）
            * 磁铁材料固有磁性弱
            * 磁铁尺寸小
            * 磁铁已退磁（or）
            * 温度过高
            * 强磁场干扰
            * 机械冲击
          * 白板磁性弱（or）
            * 白板材料非铁磁性
            * 白板表面涂层厚
            * 白板背板非磁性材料
          * 磁铁与白板之间的距离大（or）
            * 纸张厚度大
            * 磁铁与白板之间有异物
            * 磁铁表面不平整
            * 白板表面不平整
          * 磁铁方向不正确（or）
            * 磁铁磁极方向错误
            * 磁铁倾斜
        * 纸张厚度大
        * 磁铁与白板接触不良（or）
          * 磁铁数量少
          * 磁铁位置不当
          * 磁铁与白板之间存在间隙
      * 纸张与白板之间的摩擦系数小（or）
        * 纸张表面光滑
        * 白板表面光滑
        * 表面有油污
        * 表面有灰尘
    * 纸张重力大
      * 纸张质量大
        * 纸张尺寸大
        * 纸张克重高
  * 磁铁从白板上脱落（or）
    * 磁铁与白板之间的磁力小（上方已分析）
    * 磁铁受到外力作用（or）
      * 风力作用
      * 人为碰撞
      * 振动
    * 磁铁与白板接触不良（上方已分析）
