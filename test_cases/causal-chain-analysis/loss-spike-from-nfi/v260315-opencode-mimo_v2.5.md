# 因果链分析报告

## 初始缺点

训练神经网络时，loss 下降到极低后突然飙升（Slingshot 尖峰），导致训练不稳定。

## 因果链分析

- 训练 loss 值骤增（Slingshot 尖峰）
  - 正确类 softmax 概率骤降
    - 正确类 logit 值骤降（从远高于其他类骤降至远低于其他类）
      - 一次或少数几步内参数的破坏性更新幅度过大（×）
        - Adam 优化器的有效步长因子偏大
          - Adam 的二阶矩估计 v_t 衰减至接近零
            - （长时间内）各参数梯度值接近零
              - Softmax Collapse 发生
                - 正确类梯度计算值等于零
                  - log-sum-exp 计算中正确类梯度贡献被舍入为零（×）
                    - 正确类 logit 值远大于其他类 logit 值
                      - 正确类 logit 值持续增大
                        - 梯度下降迭代步数累积多
                    - float32 精度不足以在 log-sum-exp 中区分 exp(z_correct) 与 sum(exp(z_all))
                      - float32 mantissa 只有 23 位
                - 权重均值 WG 发生偏移
                  - 损失函数对权重的零和约束不成立
                    - SC 期间正确类梯度计算值等于零（上方已分析）
              - （高置信时）错误类的梯度值也接近零
        - SC 打破时单步梯度的 L2 范数大
          - SC 打破时模型参数处于不稳定配置（WG 与特征均值 µG 反平行）
            - WG 向 µG 的反方向偏移（零和约束不成立期间的累积效果）
              - 零和约束不成立期间 WG 无归零约束力（上方已分析）
            - 参数向量的 L2 范数已剧烈增长（SC 期间正反馈放大）
              - WG 与 µG 反平行条件下梯度指向进一步增大参数范数的方向
