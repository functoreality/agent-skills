# 因果链分析报告

## 初始缺点
训练loss突然大幅上升（Slingshot尖峰现象）

## 因果链分析

- 训练loss突然大幅上升
  - 参数更新后的参数值位于有效解区域之外
    - 参数位移量远大于有效解区域的尺度
      - 该步参数更新的位移量约为正常训练步的50倍
        - Adam优化器的有效步长远大于正常训练步的有效步长
          - Adam参数更新量为η·m/(√v+ε)
          - v趋近于零时分母趋近于ε
          - 有效步长近似为η·m/ε
        - v（梯度二阶矩的指数移动平均）趋近于零
          - β₂小于1（默认0.999）
            - PyTorch Adam优化器的默认参数设置为β₂=0.999
          - g_t²持续接近零
            - SC期间总梯度模长接近零
              - 正确类别的cross-entropy梯度在float32中精确为零
                - 正确类别的softmax概率在float32中精确为1.0
                  - log-sum-exp的计算值在float32中等于z_c
                    - 1.0与Σ_{j≠c} exp(z_j - z_c)之和在float32中等于1.0
                      - Σ_{j≠c} exp(z_j - z_c)的值小于float32尾数的最低精度位
                        - z_c（正确类别logit）远大于所有z_j（其他类别logit）
                          - 训练后期模型对正确类别的置信度接近1
                            - 正确类别权重向量与特征的内积远大于其他类别
                              - 模型在训练集上充分拟合
                                - 训练持续进行
                          - NFI下权重模长持续增长
                            - （单独深入分析：Numerical Feature Inflation）
                        - float32尾数仅有23位（含隐含位共24位有效位）
                          - IEEE 754标准定义float32格式为1位符号+8位指数+23位尾数
                - 正确类别的one-hot标签值为1
                  - 分类任务的标签编码方式为one-hot
                - cross-entropy的梯度值等于softmax概率减去目标值
                  - 交叉熵损失函数的数学定义为L=-Σ y_j log(p_j)
              - 其他类别的softmax梯度为接近零的正数
                - 其他类别的softmax概率为接近零的正数
                  - 各个exp(z_j - z_c)的值为接近零的正数（float32中非零）
                    - exp(z_j - z_c)的值高于float32的最小正数表示下界（约1.2×10^(-38)）
                      - z_j - z_c的值未低至float32下溢阈值（约-88）
                      - SC的触发阈值（约16.6）远低于float32下溢阈值（约88）
          - SC持续步数足够多
            - v的指数移动平均衰减为渐进过程
        - ε_Adam取默认值1e-8
          - PyTorch的Adam优化器默认参数设置为ε=1e-8
        - η（学习率）为正常量级（如1e-3）
          - 训练配置的预设值为η=1e-3
        - 该步梯度的一阶矩估计非零
          - SC效应在该步对部分训练样本失效
            - 部分样本的正确类别logit与其他类别logit的差距回落到float32精度阈值以下
              - NFI的指数增长改变了各样本的logit结构
                - （见NFI分析）
              - 个别样本的特征在NFI漂移后与正确类别权重方向的对齐度降低
                - 权重均值WG的持续漂移改变了各类权重向量的有效方向
                  - WG的模长增长在各类权重中添加了共模偏移分量
                - 部分样本的特征偏差方向对正确类别logit的贡献为负
                  - 个别样本的特征偏离方向与-WG方向的夹角较大
    - 有效解区域的尺度有限
      - 损失函数曲面在极小值附近曲率有限
  - 参数更新前的参数值位于有效解区域内
    - 训练loss在spike前已降至极低

## 周期性说明

Slingshot以循环方式反复出现：loss尖峰后模型继续训练，loss缓慢恢复至极低水平，模型重新进入高置信阶段，SC再次触发，NFI再次发展，最终再次产生尖峰。整个机制的因果结构在每次循环中相同。

# 单独深入分析：Numerical Feature Inflation（NFI）

- NFI下权重模长持续增长
  - 分类器权重均值WG沿特征均值µG的反方向漂移
    - cross-entropy梯度的零和约束不成立
      - 精确算术下softmax梯度对各类别之和为零
        - softmax概率之和恒为1
        - one-hot标签之和恒为1
        - 梯度按类别求和为零（梯度 = softmax - one_hot）
      - SC下正确类梯度为零
      - SC下其他类梯度为正
        - 梯度之和 = Σ_{j≠c} softmax(z_j) > 0
    - 梯度在各类上的均值非零
      - 各类权重更新量与该类梯度成正比
      - WG的更新量非零
    - WG偏移方向与µG相反
      - SC下仅错误类权重更新（正确类梯度为零）
      - 错误类权重更新方向为沿特征的反方向（Δw_j = -η·softmax(z_j)·x）
      - 各错误类权重更新的均值沿-µG方向
        - 训练样本特征均值为µG
        - softmax(z_j)为正权重
  - 特征均值µG沿WG的反方向漂移
    - 损失函数的梯度经反向传播传递到特征提取器
      - 特征提取器接收到的梯度缺失正确类分量
    - 特征更新方向沿-WG方向偏移
      - 损失对特征的梯度方向在SC下近似为Σ_{j≠c} softmax(z_j)·w_j
      - 此梯度为各错误类权重向量的正权重线性组合
      - 错误类权重向量的均值近似为WG
      - 特征沿负梯度方向（近似-WG方向）调整
  - WG与µG互相放大形成正反馈（+）
    - WG沿-µG方向偏移（×）
      - µG沿-WG方向偏移（循环因果）
    - 反平行方向上的分量持续增大
  - WG与µG的模长指数增长
    - 偏移速率与当前偏移量近似成正比
      - 指数增长率由学习率和梯度特性决定
    - WG与µG趋向反平行
  - NFI对logit结构产生双重影响
    - 权重模长增长放大所有类别的logit绝对值（扩大logit差距，加剧SC）
      - 各类权重向量与特征的内积绝对值增大
    - WG与µG的反平行增长改变了logit的相对差异（最终终结SC）
      - 共模偏移分量对各类logit的贡献相同
      - 残差分量为（w_i - WG）与（f(x) - µG）的内积
      - 残差分量随NFI漂移而变化
      - 个别样本的残差分量变化减小了正确类别logit
