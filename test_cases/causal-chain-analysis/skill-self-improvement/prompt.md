```markdown
- **精确区分**：务必区分非常相似、但实际上不同的概念。比如，`参与注意力计算的元素个数`、`输入数据总 token 数`、`输入文本单词数/图像像素个数/视频帧数`，是三个完全不同的东西，绝不能用笼统的`序列长度`糊弄过去。
```
