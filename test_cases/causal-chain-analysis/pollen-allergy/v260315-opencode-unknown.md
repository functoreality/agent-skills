# 因果链分析报告

## 初始缺点
眼部结膜组织对花粉过敏原产生过敏反应

## 因果链分析

- 眼部结膜组织对花粉过敏原产生过敏反应
  - 过敏原（花粉）接触眼部结膜
    - 空气中的花粉浓度过高
      - 春季植物大量开花
        - 春季气温适宜植物生长
          - 春季日照时间逐渐增长
          - 春季积温逐渐回升
        - 春季空气干燥多风
          - 春季降水量相对较少
          - 春季大气垂直运动活跃
      - 花粉扩散条件充足
        - 风力促进花粉传播
          - 春季多大风天气
          - 花粉颗粒质量轻易被风卷起
        - 城市绿化带分布广泛
          - 城市园林植物种类多
          - 柏科杨柳科植物种植面积大
    - 花粉颗粒能够进入眼部
      - 眼部无有效防护
        - 未佩戴护目镜或防护眼镜
      - 花粉在眼表积聚
        - 眼泪对花粉的冲刷能力有限
        - 眼睑眨动不能完全清除花粉
  - 个体免疫系统对花粉产生过度反应
    - 过敏体质人群的IgE水平偏高
      - 遗传因素导致特应性体质
      - 既往过敏原暴露致敏
    - 花粉中的特定蛋白成分被识别为有害物质
      - 花粉含有可溶性蛋白
      - 蛋白质结构具有免疫原性
    - 肥大细胞释放组胺等炎性介质
      - IgE与肥大细胞表面FcεRI受体交联
        - 肥大细胞胞膜受体聚集
        - 钙离子内流触发脱颗粒
      - 组胺与结膜血管、神经末梢受体结合
        - H1受体介导血管扩张
        - 神经纤维受刺激引发瘙痒感
  - 结膜部位存在大量肥大细胞和嗜碱性粒细胞
    - 结膜黏膜富含免疫细胞
      - 结膜淋巴组织分布丰富
      - 黏膜相关淋巴组织存在
    - 结膜直接暴露于外界环境
      - 结膜表面缺乏物理屏障
      - 结膜血管网密集

- 眼部结膜瘙痒感强烈
  - 组胺激活结膜的感觉神经末梢
    - 结膜分布有丰富的无髓鞘C纤维
    - 神经纤维表面有组胺受体
  - 神经信号传导至大脑
    - 三叉神经眼支传导信号
    - 中枢神经系统处理瘙痒感
  - 搔抓反射和行为
    - 眼部不适引发揉眼动作
    - 揉眼进一步刺激结膜

- 需要佩戴护目镜才能出门
  - 不佩戴护目镜时花粉直接接触眼部
    - 花粉在眼表持续存在
    - 过敏反应持续进行
  - 佩戴护目镜可有效阻隔花粉
    - 护目镜形成物理屏障
    - 减少过敏原与结膜接触

- 结膜炎（过敏性结膜炎）发作
  - 结膜炎症反应持续
    - 炎性细胞浸润
    - 血管通透性增加导致水肿
  - 症状表现为眼痒、流泪、充血
    - 眼痒程度剧烈
    - 眼部充血发红
    - 眼泪分泌增多
