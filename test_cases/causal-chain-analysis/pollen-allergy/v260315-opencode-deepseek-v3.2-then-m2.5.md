# 因果链分析报告

## 初始缺点

眼睛瘙痒感强烈

## 因果链分析

- 眼睛瘙痒感强烈
  - 结膜炎炎症反应强烈（+）
    - 眼睛发生过敏反应
      - 过敏原接触眼睛黏膜
        - 花粉接触眼睛黏膜
          - 空气中花粉浓度高
            - 北京春天花粉产生量高
              - 春季树木进入开花期
                - 柏科植物花粉量大
                - 杨柳科植物进入飞絮期
              - 春季花卉进入盛开期
                - 蔷薇科植物花粉量大
                - 木犀科植物花粉量大
              - 气候条件适合花粉传播
                - 气温回升快
                - 空气湿度低
                - 风力适中至偏大
          - 花粉沉积于眼表
            - 眼睛暴露在空气中
              - 使用者未佩戴护目镜
            - 眨眼频率不足
              - 泪液清除能力有限
                - 泪膜分布不均匀
                - 泪液蒸发过快
      - 免疫系统过度反应
        - 免疫系统对花粉过敏
          - 花粉作为过敏原
            - 个体为过敏体质
            - 存在遗传因素
            - 存在既往过敏史
              - 之前花粉季节出现过症状
              - 存在其他过敏性疾病史
    - 眼睛发生非过敏性炎症（or）
      - 细菌或病毒感染
      - 其他刺激物作用
  - 神经末梢受刺激（+）
    - 瘙痒感受器被激活
      - 组胺等炎症介质作用于神经末梢
        - 组胺释放增加
        - 组胺与H1受体结合
        - 信号传导至大脑产生瘙痒感
