# 因果链分析报告

## 初始缺点

书本页脚发生塑性弯曲变形（卷弯），导致需要压很久才能恢复平整。

## 因果链分析

- 书本页脚发生塑性弯曲变形（×）
  - 弯曲应力大（×）
    - 弯矩大（×）
      - 书页重力大
        - 书页质量大（×）
          - 书页密度大
          - 书页体积大（×）
            - 书页面积大
      - 力臂长度大
        - 页脚到支撑点的距离大
          - 书本高度大
          - 支撑点在书本底部
    - 截面模量小（×）
      - 书页宽度小
      - 书页厚度小
  - 材料屈服强度低
    - 纸张材料本身屈服强度低
      - 纸张纤维结合力弱
      - 纸张含水量高
        - 环境湿度高
        - 纸张亲水性强
  - 作用时间长
    - 书本竖放时间长
      - 书包携带时间长
      - 书本竖放频率高
  - 材料蠕变敏感性高
    - 纸张纤维素蠕变特性明显
