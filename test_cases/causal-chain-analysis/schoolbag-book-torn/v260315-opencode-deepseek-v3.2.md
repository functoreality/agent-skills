# 因果链分析报告

## 初始缺点
书本页面卷边或折角。

## 因果链分析

* 书本页面卷边或折角（or）
  * 书本页面卷边
    * 页面边缘弯曲形变
      * 页面边缘受到弯曲力矩（or）
        * 外部压力作用于页面边缘
          * 书本在包内受到挤压
            * 包内物品过多
            * 包内空间狭小
        * 膨胀应力存在
          * 页面边缘受到约束
          * 页面内部存在湿度梯度
            * 纸张吸湿膨胀系数不均
            * 环境湿度变化
              * 包内湿度变化
                * 外界湿度变化
                * 人体汗液蒸发
            * 纸张干燥不均匀
      * 页面材料塑性变形
        * 弯曲应力超过纸张弹性极限
          * 弯曲力矩过大
          * 纸张弹性模量低
            * 纸张受潮
              * 环境湿度高
              * 纸张接触液体
          * 纸张截面模量小
        * 纸张发生蠕变
          * 应力作用时间长
            * 书本在包内放置时间久
          * 纸张蠕变率高
  * 书本页面折角
    * 页面局部弯曲形变
      * 页面受到集中弯曲力（or）
        * 页面被其他物品顶弯
          * 包内尖锐物品压迫页面
          * 书本与其他物品摩擦
        * 页面被手指折弯
          * 取放书本时页面受到弯曲力
        * 页面被外力折弯
          * （上方已分析）
      * 页面材料塑性变形
        * 弯曲应力超过纸张弹性极限
          * 弯曲力过大
          * 纸张弹性模量低（上方已分析）
          * 纸张截面模量小
        * 纸张发生蠕变
          * 应力作用时间长
            * 书本在包内放置时间久
          * 纸张蠕变率高
