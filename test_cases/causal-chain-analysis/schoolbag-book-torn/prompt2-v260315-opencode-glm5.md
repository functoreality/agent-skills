# 因果链分析报告

## 初始缺点

书页下角呈现弯曲变形状态

## 因果链分析

- 书页下角呈现弯曲变形状态
  - 书页下角的纸纤维发生非均匀塑性变形（×）
    - 书页下角受到弯曲应力
      - 书页下角受到弯曲力矩（×）
        - 外力作用于书页下角（+）
          - 书包内物品挤压书页下角
            - 书包内放置其他物品
            - 书包内空间有限
          - 书包内壁对书页下角施加压力
            - 书包柔性材料随运动形变
              - 书包材料为柔性织物
            - 书包受力时内壁贴近书页
          - 书本自身重量对下角产生压力
            - 书本有一定重量
            - 书本竖放时底部承重
          - 书页缺少外部保护结构
            - 书本未使用书套或护角
        - 外力作用点与书页装订边之间存在力臂
          - 书页下角突出于书本主体之外
            - 书本竖放时书页在重力作用下向外散开
              - 书页之间无紧固约束
              - 重力作用于松散的书页边缘
            - 书页自由端与装订点距离大
              - 书页装订点位于书脊
              - 书页长度大
          - 书页近装订处受到约束
            - 书芯主体对相邻书页提供支撑
            - 书皮对内侧书页提供约束
      - 书页具有抗弯刚度
        - 纸张材料具有抗弯性能
        - 弯曲时纸张内外侧存在应力差
          - 纸张具有一定厚度
          - 纸张弯曲时一侧受拉一侧受压
    - 弯曲应力多次超过纸的弹性极限（×）
      - 弯曲应力超过纸的弹性极限（×）
        - 弯曲应力幅值足够大
          - 外力幅值大
          - 力臂足够长
        - 纸的弹性极限低（or）
          - 纸张本身弹性极限较低
            - 纸张材料弹性极限由材料成分决定
          - 纸张处于吸湿状态
            - 纸张吸湿
              - 环境空气湿度高
              - 纸张纤维具有吸湿性
                - 纤维素分子含有亲水基团
          - 纸张处于高温状态
            - 温度升高
              - 环境温度高
              - 书包内温度随外界变化
      - 弯曲应力多次循环作用
        - 外力反复作用（+）
          - 行走时书包晃动
            - 人体持续行走
            - 书包随身体运动
          - 书包内物品位置反复变化
            - 书包柔性外壁随运动形变
            - 其他物品在包内移动
          - 书页之间相互摩擦
            - 书页之间存在相对运动趋势
            - 书页表面存在粗糙度
              - 造纸工艺形成纸张表面微观凹凸
        - 书本在书包中放置时间足够长
