# 因果链分析报告

## 初始缺点
文档外观受损（页面卷边或折角）

## 问题背景
用户将书本或笔记本放在包里，经过一段时间后，页面出现卷边或折角现象，影响文档的美观和使用体验。

---

## 因果链分析

- 文档页面卷边或折角（or）
  - 页面发生弯曲变形（卷边）
  - 页面发生折叠变形（折角）

### 一、页面卷边（弯曲变形）

- 页面发生弯曲变形
  - 页面受到弯矩作用（×）
    - 力作用于页面
      - 外部压力作用于页面（or）
        - 包内物品挤压页面
          - 包内物品数量多
          - 包内空间有限
            - 包的容量有限
            - 放入的物品体积大
        - 包受到外部压力
          - 身体挤压包
            - 人坐姿挤压包
            - 包放置于狭窄空间受压
        - 页面处于被夹持状态
          - 多本书/物品层叠
      - 包在运动中震荡摇晃
        - 人背着包移动
          - 人行走或运动
          - 包的固定方式不稳固
        - 运动产生动态载荷
          - 包内物品晃动
          - 惯性力作用于物品
    - 力臂存在
      - 页面部分区域未受支撑
        - 书本部分悬空
          - 书本装订边缘外露
          - 书本放置位置不平整
        - 书页之间存在间隙
          - 书页贴合不紧密
          - 空气夹层存在于书页间
      - 物品局部施力于页面边缘
        - 物品棱角压在页面边角
          - 物品形状不规则
          - 物品边缘尖锐
  - 弯曲产生的应力超过纸张的弹性极限
    - 弯曲产生的应力大
      - 弯曲曲率大
        - 施加的弯矩大
          - 力大（见上方分析）
          - 力臂大（见上方分析）
        - 弯曲变形空间小
          - 书本被挤压空间有限
      - 纸张局部受力面积小
        - 物品边缘或棱角接触页面
    - 纸张弹性极限低
      - 纸张材料刚度较低
        - 纸张纤维密度低
        - 纸张纤维结构疏松
      - 环境因素降低纸张强度
        - 环境湿度高
          - 纸张吸收水分
            - 纸张纤维细胞壁中的纤维素和半纤维素具有亲水性
            - 纸张暴露在潮湿环境中
              - 空气湿度高
              - 包内透气性差导致湿气积聚
        - 环境温度高
          - 高温加速纸张老化
      - 纸张存在初始损伤
        - 纸张边缘有微裂痕
        - 纸张局部磨损
  - 弯曲变形维持时间足够长
    - 书本在包内放置时间长
      - 用户长时间不取用
    - 变形在较长时间内无法恢复
      - 塑性变形持续时间长
      - 纸张蠕变特性导致变形累积
        - 纸张纤维在持续载荷下发生蠕变
          - 纸张纤维之间的氢键发生重排
            - 纤维素分子链在应力作用下滑移

### 二、折角（折叠变形）

- 页面发生折叠变形
  - 页面受到剪切力或局部压力
    - 包内物品相对运动产生剪切
      - 物品之间发生相对滑动
        - 堆叠的物品界面存在压力
          - 包内空间拥挤
        - 界面摩擦系数不足以阻止滑动
          - 界面光滑
          - 运动加速度大
      - 页面边缘或角部被其他物品推挤
        - 物品摆放位置使边角外露
          - 书本未完全放入包内
          - 书本尺寸与包内空间不匹配
        - 其他物品角部或边缘尖锐
    - 局部压力集中于页面边角
      - 物品棱角压在页面上
        - 硬质物品（如笔盒、电子设备）棱角尖锐
        - 书本在包内倾斜使边角受压
      - 页面边角暴露在受力区域
        - 书本未对齐放置
        - 包内物品排列不规则
  - 页面材料在应力集中处屈服
    - 局部应力超过纸张屈服强度
      - 应力集中
        - 几何形状突变（如尖角）
        - 接触面积小
      - 纸张屈服强度低
        - 纸张材料弹性模量低
          - 纸张纤维密度低
          - 纸张纤维排列疏松
        - 环境因素降低纸张强度
          - 环境湿度高
            - 纸张吸收水分
              - 纸张纤维细胞壁中的纤维素和半纤维素具有亲水性
              - 纸张暴露在潮湿环境中
                - 空气湿度高
                - 包内透气性差导致湿气积聚
          - 环境温度高
            - 高温加速纸张老化
        - 纸张存在初始损伤
          - 纸张边缘有微裂痕
          - 纸张局部磨损
  - 折痕形成并固定
    - 折叠线处纤维发生塑性变形
      - 纤维断裂或严重弯曲
      - 纤维间氢键断裂后重新形成
    - 折痕不可逆
      - 纸张缺乏弹性恢复能力

## 因果链结构说明

### 分析层级说明

本分析共到达约 **10-12 层**深度，识别出以下主要因果路径：

1. **卷边路径**：外部压力/震荡 → 弯矩 → 应力 → 超过弹性极限 → 塑性变形 → 变形累积
2. **折角路径**：剪切力/局部压力 → 应力集中 → 屈服 → 纤维断裂 → 折痕固定

### 关键末端缺点

- 物品边缘或棱角接触页面
- 包内空间有限（系统设计决策）
- 纸张纤维密度低
- 纸张具有亲水性（纤维特性）
- 纸张纤维在持续载荷下发生蠕变
- 纤维素分子链在应力作用下滑移
- 纤维间氢键断裂后重新形成
- 人背着包移动（自然行为）
- 用户长时间不取用（人的行为）

### 可干预的关键中间缺点

- 包内物品数量多
- 放入的物品体积大
- 包受到外部压力（如坐下时挤压）
- 环境湿度高
- 环境温度高
- 书本尺寸与包内空间不匹配
- 书本未对齐放置
- 包内物品排列不规则
- 硬质物品棱角尖锐

---

## 可能的解决方案映射

| 解决方案 | 对应的末端/中间缺点 |
|---------|-------------------|
| 使用硬壳文件夹或文件袋 | 物品边缘接触页面、局部压力集中 |
| 减少包内物品数量 | 包内物品数量多、空间拥挤 |
| 选择合适尺寸的包 | 包内空间有限 |
| 使用防潮袋保持干燥 | 环境湿度高 |
| 避免久坐挤压包 | 包受到外部压力 |
| 整理包内物品排列 | 物品排列不规则、边角外露 |
| 使用书皮保护 | 局部压力集中于边角 |
| 选择更高质量纸张的书本 | 纸张纤维密度低、弹性极限低 |
