# 因果链分析报告

## 初始缺点

气动软体机器人制动器弯曲角度小

## 因果链分析

- 气动软体机器人制动器弯曲角度小（or）
  - 驱动制动器弯曲的力矩小
    - 气室内部气体作用在壁面上的力小
      - 气室内壁有效受力面积小
        - 气室沿弯曲方向的投影面积小
          - 气室在弯曲方向的跨度小
          - 气室在垂直弯曲方向的宽度小
        - 气室内壁面积被非受力结构占据
          - 气室内部有支撑结构
          - 气室内壁有加强筋
      - 气室内部气压低
        - 气源供气压强低
          - 气源压力上限低
          - 气压调节阀设定值低
        - 气室内部气压损失速率大
          - 气体从侧壁缝隙漏出
            - 侧壁存在贯通缝隙
              - 打印路径在侧壁层形成间隙
                - 打印喷头在层间移动时材料中断
                  - 打印喷头执行回抽操作
                    - 挤出机向后拉动耗材
                      - 控制器发出回抽指令
                        - 切片软件生成回抽指令
                          - 切片软件检测到非连续打印路径
                            - 打印路径存在跳跃段
                              - 打印件几何形状包含分离区域
                                - 气室结构包含封闭空腔
                              - 打印件有悬空结构
                            - 回抽功能开关处于开启状态
                    - 喷嘴腔体内材料被抽回
                      - 喷嘴腔体与耗材通道连通
                      - 耗材与喷嘴腔内材料连接
                  - 喷头移动速度过快
                    - 移动速度超过材料可跟随速度
                      - 移动段速度设定值高
                        - 切片软件移动速度参数高
                    - 材料局部应变率大
                      - 材料在短距离内被快速拉伸
                        - 喷头移动距离短
                        - 喷头移动时间短
                      - 粘弹性材料在高应变率下呈现脆性断裂行为
                        - 材料存在粘弹性
                          - TPU具有高弹性
                            - TPU分子链可伸展
                        - 高应变率下分子链来不及伸展
                          - 拉伸作用时间短于分子链松弛时间
                            - TPU分子链松弛时间长
                              - TPU分子量大
                              - TPU分子链缠结程度高
                                - TPU分子链具有柔性结构
                                  - TPU软段分子链柔顺性好
                  - 新位置开始挤出时材料供应滞后
                    - 回抽后材料充满喷嘴的时间长（×）
                      - 喷嘴腔体需要填充的体积大
                        - 回抽操作使喷嘴腔体部分空虚
                          - 回抽距离大于零
                      - 熔融材料进入喷嘴的速率有限
                        - TPU熔体粘度高
                          - TPU分子量大（上方已分析）
                          - TPU分子链缠结程度高（上方已分析）
                        - 喷嘴内径小
                          - 打印层高小
                          - 打印线宽小
                        - 挤出驱动力有限
                          - 进料齿轮与TPU之间的摩擦力不足
                            - 接触面压强低
                              - 进料齿轮压簧力小
                            - TPU表面摩擦系数低
                              - TPU材料表面光滑
                                - TPU硬度低时表面易变形贴合
                          - 耗材在进料管内阻力大
                            - 进料管内径与耗材间隙小
                            - 进料管弯折
                              - 进料路线存在弯角
              - 层间间隙在壁厚方向上贯通
                - 多层间隙在壁厚方向上对齐或部分重叠
                  - 相邻层的打印路径相似
                    - 每层打印路径由同一切片算法生成
                      - 切片算法沿层高方向扫描模型
                  - 层间间隙位置相近
                    - 模型几何在层间变化小
                      - 气室壁倾角小
              - 间隙处无材料填充
                - 后续挤出材料未填补间隙
                  - 间隙形成时喷头已移走
                    - 间隙形成后喷头执行移动指令
                  - 间隙处材料温度不足以熔合
                    - 材料在喷头移动期间散热
                      - 材料与环境存在温差
                        - 打印环境温度低于材料熔点
                          - 环境温度为室温
                          - TPU材料熔点高于室温
                      - 材料散热表面积相对体积大
                        - 细丝或薄壁结构散热快
                    - 喷头移动过程中不提供热量
                      - 喷头在移动时不挤出材料
                        - 移动指令期间挤出机停止工作
            - 间隙两端存在压差
              - 气室内部气压高于外部
                - 气室充气时内部气压升高
                  - 外部气源向气室供气
              - 外部气压为大气压
                - 制动器在空气中工作
            - 间隙对气体的流动阻力低
              - 间隙有效流通截面大
                - 单个间隙的开口大
                  - 材料中断形成较大开口
              - 间隙流道长度短
                - 壁厚方向间隙贯通距离短
                  - 壁厚小
                    - 壁厚度设计值小
    - 驱动力作用点到弯曲中性轴的距离小
      - 气室中心到制动器截面中性轴的距离小
        - 气室在制动器截面中的位置靠近中性轴
          - 气室布置在制动器截面中性层附近
            - 制动器截面设计为对称结构
              - 气室位于截面对称轴上
        - 制动器截面的中性轴位置靠近气室
          - 制动器上下壁厚接近
  - 制动器结构对弯曲变形的等效刚度大
    - 制动器壁的抗弯刚度大（×）
      - 制动器壁材料弹性模量大
        - TPU材料的杨氏模量大
          - TPU硬度等级高
            - TPU配方中硬段含量高
              - 硬段质量占比大
                - 硬段分子链刚性大
                  - 硬段含芳香族结构
          - 可选TPU材料硬度范围受限
            - 软TPU打印性能差
              - 软TPU易堵塞喷嘴
                - 软TPU在高温下流动不稳定
                  - 软TPU软段比例高
              - 软TPU进料困难
                - 软TPU在进料管内弯折
                  - 软TPU材料刚度低
      - 制动器壁截面惯性矩大
        - 制动器壁厚度大（三次方关系）
          - 壁厚度设计值大
        - 制动器壁截面形状抗弯能力强
          - 壁截面为实心矩形
          - 壁截面上材料分布远离中性轴
    - 制动器非壁结构的等效弯曲刚度大
      - 气室之间隔膜的等效弯曲刚度大
        - 隔膜厚度大
          - 隔膜厚度设计值大
        - 隔膜材料的弹性模量大
          - 隔膜由TPU材料打印
      - 气室周围加强结构的等效弯曲刚度大
        - 加强筋的数量多
        - 加强筋的截面面积大
          - 加强筋厚度大
          - 加强筋宽度大
    - 制动器结构几何布局不利于单一方向弯曲
      - 壁厚沿弯曲方向分布不均匀
        - 局部壁厚度大
        - 局部壁厚度小
      - 气室排列方向与期望弯曲方向不一致
        - 气室布局方向偏离最优弯曲方向
          - 整体结构空间限制
  - 制动器有效弯曲长度短
    - 单个气室沿弯曲方向的长度短
      - 气室长度设计值小
    - 参与弯曲变形的气室数量少
      - 制动器沿弯曲方向布置的气室数量少
        - 气室数量设计值小
    - 制动器两端刚性固定段占用长度大
      - 进气口端固定结构长
      - 末端固定结构长
