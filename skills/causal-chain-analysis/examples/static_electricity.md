# 因果链分析示例：静电打到手

- 手指感到疼痛
	- 手指的神经末梢痛觉感受器丰富
	- 电流通过手指
		- 电压存在
			- 电荷存在
				- 电荷产生
					- 摩擦产生电荷
						- 物体发生摩擦
							- 两物体接触
							- 两物体相对运动
						- 材料性质容易摩擦起电
				- 电荷不能导出
					- 衣物不导电
						- 衣物材料不导电
						- 衣物干燥
					- 空气不导电
						- 空气干燥
		- 回路形成
			- 手触碰金属门把手
		- 手是导体
		- 门把手是导体
	- 神经信号传递到大脑
